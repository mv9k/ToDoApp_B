About



Step 1: Add a task by entering in the list name it goes under and the name of the task.

Each task you add requires a list name to go under and a name for the task.

Step 2: Navigate through multiple lists with the list tabs that appear at the top.

Here you can view all lists, individual lists, see how many tasks are finished out of the total tasks for each list, as well as clear all completed tasks.

Step 3: Finish tasks with the checkmark button, or delete them with the X button. You can also rename a task by clicking it's name.

The checkmark button can be clicked again on a finished task to mark it as unfinished. You can clear finished tasks with the button at the top next to the list tabs. To delete a list, just click the X button next to the list name. This will also delete all tasks corresponding to that list.
